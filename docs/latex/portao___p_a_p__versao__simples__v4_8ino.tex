\doxysubsection{Detailed Description}
Controle de portões. 

Este código foi criado para controlar a abertura de dois portões miniatura como parte do PAP para 12 alunos do TMI, em 2024.

Criado em maio de 2024 por Olivares, revisado por Sarg. Freitas.
